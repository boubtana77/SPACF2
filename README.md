# SPACF

SPACF (Search • Prospect • Acquire • Convert • Flow) is a lean internal AI-powered client acquisition engine for finding local businesses with no, weak, or outdated websites, prioritizing the best opportunities, and running practical outreach workflows.

## Focus

- Global business discovery by country, location, and category.
- Website-status filters for no website, weak website, and old website prospects.
- AI opportunity scoring for HOT, WARM, and COLD leads.
- Lightweight CRM pipeline from New to Closed.
- Manual and automated outreach sequences with AI-generated email styles.
- Reply analysis with suggested response, pricing approach, and next step.
- Simple analytics for outreach and conversion performance.
- Reusable niche demo landing page assets for faster conversion.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL / Supabase-ready API architecture

## Local development

```bash
npm install
npm run dev
```
